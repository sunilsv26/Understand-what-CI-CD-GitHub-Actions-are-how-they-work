# Express setup

This repository now includes a minimal Express server and startup scripts.

Quick start

1. Install dependencies:

```powershell
npm install
```

2. Start server:

```powershell
npm start
```

3. For development with auto-reload (requires `nodemon`):

```powershell
npm run dev
```

Endpoints

- `GET /` — returns a welcome JSON message.
- `GET /health` — returns a simple health JSON.
